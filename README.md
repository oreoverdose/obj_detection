# obj_detection

## Dependencies
* cv_bridge
* image_transport
* std_msgs
* rospy

## Installation
1. Set up a catkin workspace
2. Clone this repository to your workspace src folder:
```
cd <catkin_ws>/src
git clone https://github.com/oreoverdose/obj_detection.git
```
3. Make sure everything is initialized:
```
cd <catkin_ws>
catkin_make
source devel/setup.bash
```
